# Mattermost Plugin CircleCI

## Summary
CircleCI Plugin For Mattermost.
Enable webhooks for CircleCI builds.
Available Slash Commands:
  - /circleci connect
  - /circleci disconnect
  - /circleci me
  - /circleci projects
  - /circleci recent builds
  - /circleci build
